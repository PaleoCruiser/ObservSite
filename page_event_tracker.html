Page Event Tracker Demo

This page demonstrates how to track key HTML document lifecycle events and send the data to a server using a simple image URL.

How it works:

  • The script logs the `DOMContentLoaded` and `load` event timestamps.
  • When you close this tab or navigate away, the `beforeunload` event will fire.
  • The script will then build a query string with the collected event times and append it to an image URL.
  • Finally, it will log the complete URL to your browser's developer console.

To see the result, open your browser's developer tools (F12 or Cmd + Opt + I) and look at the "Console" tab after you leave the page.

This is a static demo. In a real application, the image request would be sent to a server that logs the data.

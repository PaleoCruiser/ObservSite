Extended Page Event Tracker Demo

This page demonstrates tracking additional, more advanced events to better understand page performance and user interaction.

New Events Tracked:

  • `visibilitychange`: Tracks when the page becomes visible or hidden (e.g., when the user switches tabs).
  • `pageshow`: Fires when the page is loaded, including when it's restored from the browser's back/forward cache. The `persisted` property is logged to show if it came from the cache.
  • `pagehide`: Fires when the user navigates away from the page. Useful for tracking how long a user spends on a page.
  • `beforeunload`: Fires before the user leaves the page. This is our last chance to send the collected data.

To see the results, open your browser's developer tools (F12 or Cmd + Opt + I) and look at the "Console" tab as you interact with the page.

This is a static demo. In a real application, the image request would be sent to a server that logs the data.
